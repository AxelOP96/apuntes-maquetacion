Cada vez que se usa VAR se asigna un espacio extra de memoria.

Undefined: No guarda ningún tipo de dato, no tiene valor

Null: Guarda una negación.

NaN: Devuelve un valor que no es un número.

VAR

Es redeclarable y reasignable. Ocupa espacio por cada declaración.

LET

No es redeclarable, es reasignable. Puede cambiar su valor, pero ocupa un solo espacio en memoria.

CONST

No es redeclarable ni reasignable. Es inmutable y no se puede volver a declarar.

DOM: Document Object Model.
