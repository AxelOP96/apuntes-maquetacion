Desarrollo web: HTML y CSS

HTML (Hypertext Markup Language) El lenguaje de marcado de hipertexto utiliza etiquetas para definir la funcion que cumple cada elementos dentro de la pagina, identificando asi la jerarquia de los mismos para facilitar a los motores de busqueda y al navegador su correcta interpretacion. Este aspecto se conoce como carga semantica y se utiliza para la indizacion de contenidos. ejemplo: <p>parrafo</p>

Clase 4: Estilos en formularios

A la etiqueta form debemos especificarle por lo menos dos atributos, que son action="..." el cual indica que debe suceder al enviar el formulario; y method="..." que indica como debe realizar el intercambio de los datos.

GET:

Método cuya prioridad radica en la respuesta del servidor, ya que los datos se envian en texto plano y permanecen visibles en la barra de direcciones. Se utiliza principalmente en buscadores.

POST:

Método cuya prioridad está proteger los datos enviados por el usuario, ya que cifra los mismos antes de realizar el envio. Se utiliza en formularios de registro, acceso, contacto, etc.

{border-style: --- ;}

Permite modificar el estilo de los bordes del elemento.

{border: --- ;}

Permite modificar las propiedades de borde de un elemento como el tamaño, el estilo y el color.

{outline: --- ;}

Permite modificar el estilo del contorno del elemento.

REM:

REM es una unidad de medidad en CSS que se basa en el tamaño de fuente del elemento raíz (<html>). Es útil para mantener tamaños proporcionales y escalables en una página.

EM:

EM es una unidad de medidad en CSS basada en el tamaño de fuente del elemento padre. Es útil para definir tamaños relativos que se ajustan dinamicamente.

{box-shadow: --- ;}

box-shadow permite aplicar sombra el elemento contenedor.

*

API

Conjunto de instrucciones previamente programadas para facilitarnos una tarea.

<iframe>

Comparte un archivo que tiene licencia como un video, mapa, etc.

<figure>

Agrupa un conjunto de imagenes relacionadas.

<figcaption>

Epigrafe de un conjunto de imagenes.

<video>

Inserta un video que puede estar en una carpeta o de una url. Tiene atributos como controls que permite el uso de botones para usar el video, autoplay que indica que se comenzará a reproducir desde que se está en la página, muted para silenciar el video, loop para que se reproduzca de manera indefinida.
